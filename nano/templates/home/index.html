Opensource invoicing webapp. Fork the project at https://github.com/sanjeevan/nanoinvoice

  • Supports invoice payments via stripe and gocardless
  • Tax, VAT support
  • Export invoice to PDF

Backend built using using Python and Flask. Frontend built using Backbone.js and jQuery (+plugins), CSS built using SCSS
